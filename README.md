# NestInsight

NestInsight is an AI-powered dataset intelligence platform that automates:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Visualization
- Machine Learning Model Training
- Business Insights Generation
- PDF/DOCX Report Generation

## Tech Stack

- Flask
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- MySQL
- ReportLab
- Python-docx

## Features

- Upload CSV datasets
- Clean missing values and duplicates
- Generate descriptive analytics
- Create heatmaps and visual reports
- Train ML models automatically
- Download cleaned datasets and reports
